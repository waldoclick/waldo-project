  id?: number;

  title?: string;
  header?: string;
  body?: string;
  seo_title?: string;
  seo_description?: string;
  source_url?: string | null;

**2. Add `published` boolean and `source_url` to the `form` ref** — insert after `seo_description`:
```typescript
source_url: string;
published: boolean;
```
Default values: `source_url: ""`, `published: false` (new articles default to draft).

**3. Update `hydrateForm()`** — populate the new fields from props:
```typescript
source_url: props.article?.source_url || "",
published: props.article?.publishedAt != null,
```

**4. Update `handleSubmit` payload** — add `source_url` and `publishedAt`:
```typescript
source_url: form.value.source_url.trim() || null,
publishedAt: form.value.published ? new Date().toISOString() : null,
```
Include both in the payload object passed to `strapi.create` and `strapi.update`.

**5. After successful update, re-sync form state** — in the edit branch after `strapi.update`, update `form.value` to include:
```typescript
source_url: payload.source_url || "",
```

**6. Add template fields** — insert two new `form__group` blocks after the `seo_description` group and before `form__send`:

```html
<!-- Source URL -->
<div class="form__group">
  <label class="form__label" for="source_url">URL de fuente</label>
  <Field
    v-model="form.source_url"
    name="source_url"
    type="url"
    class="form__control"
    placeholder="https://..."
  />
  <ErrorMessage name="source_url" />
</div>

<!-- Draft/Publish toggle -->
<div class="form__group form__group--toggle">
  <label class="form__label--toggle" for="published">
    <input
      id="published"
      v-model="form.published"
      type="checkbox"
      class="form__toggle"
    />
    <span>{{ form.published ? 'Publicado' : 'Borrador' }}</span>
  </label>
</div>
```

**7. Add `source_url` to the yup schema** (optional, no validation):
