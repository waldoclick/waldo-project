# Phase 51: Seeders + Test Files any Elimination — Plan 1

**Plan:** 51-01
**Goal:** Eliminate all `any` types from 5 seeder files and 4 test files — seeders use `Core.Strapi`; test files access results through typed interfaces
**Requirements:** TSANY-33, TSANY-34, TSANY-35, TSANY-36
**Wave:** 1 of 1
**Estimated tasks:** 2

---

## Context

Nine files contain all the `any` types for this phase:

**Seeders (all in `apps/strapi/seeders/`):**
1. `categories.ts` — `strapi: any` in `populateCategories`
2. `packs.ts` — `strapi: any` in `populatePacks`
3. `regions.ts` — `strapi: any` in `populateRegions`
4. `faqs.ts` — `strapi: any` in `populateFaqs`
5. `conditions.ts` — `strapi: any` in `populateConditions`

**Test files:**
6. `apps/strapi/src/api/payment/services/__tests__/pack.zoho.test.ts` — `(global as any).strapi` (×2), `(result as any).success` (×2)
7. `apps/strapi/src/api/payment/services/__tests__/pack.service.test.ts` — `(await packService.packPurchase(...)) as any`
8. `apps/strapi/src/api/payment/services/__tests__/ad.service.test.ts` — `(await adService.processPaidPayment(1)) as any`
9. `apps/strapi/src/api/payment/controllers/__tests__/payment.controller.test.ts` — `body: undefined as any`, `(controller as any).packResponse` (×3)

No new packages. No new files. All changes are in-place edits.

**Key constraint:** `tsconfig.json` excludes `**/*.test.*` from compilation, so `tsc --noEmit` only verifies seeders. Test file type correctness is verified via `yarn test` (ts-jest performs type checking during test execution).

Catch block `error: unknown` narrowing is **out of scope** per REQUIREMENTS.md — do NOT change `catch (error)` parameter types.

### What to do

#### pack.zoho.test.ts (TSANY-34)

There are 4 `any` usages:
- Line 16: `(global as any).strapi = {...}` — initial mock setup
- Line 97: `(global as any).strapi.entityService.findOne.mockResolvedValue(...)` — beforeEach reset
- Line 143: `expect((result as any).success).toBe(true)` — Test 3
- Line 153: `expect((result as any).success).toBe(true)` — Test 4

**Step 1 — Add typed interface and global declaration after the mock comment block (before line 14):**

Insert immediately after the `// ─── Mock strapi global before imports ───` comment and before `(global as any).strapi = {`:

```typescript
// ─── Mock strapi global before imports ───────────────────────────────────────

interface MockStrapi {
  entityService: {
    findOne: jest.Mock;
  };
}

declare global {
  // eslint-disable-next-line no-var
  var strapi: MockStrapi;
}

global.strapi = {
  entityService: {
    findOne: jest.fn().mockResolvedValue({ email: "user@example.com" }),
  },
};
```

Remove the old `(global as any).strapi = {...}` block (lines 16–20).

**Step 2 — Fix `beforeEach` reset (line 97):**
```typescript
// BEFORE

    email: "user@example.com",
  });

// AFTER

    email: "user@example.com",
  });
```
