<objective>
Migrate all components that use `useStrapiClient()` directly for mutating HTTP calls to use `useApiClient()` instead. This includes 3 auth forms (which currently have manual `$recaptcha.execute` logic that becomes redundant) and 4 components that never had reCAPTCHA protection.

Purpose: These components make direct HTTP calls via `useStrapiClient()` with explicit `method: "POST"` or `method: "PUT"`. After the server guard broadening (Plan 01), these calls will be rejected unless they include `X-Recaptcha-Token`. Migrating to `useApiClient()` adds the token automatically.
Output: 7 components migrated, manual reCAPTCHA logic removed from auth forms.
</objective>
