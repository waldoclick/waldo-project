**In controllers/ad.ts:**
Add the `meCounts` method to the controller factory. It must:
1. Check auth: return 401 if `ctx.state.user` is missing
2. Run 5 parallel `strapi.entityService.count()` calls — one per status — using the
   same filter logic already in `me()` (copy the switch-case filter construction,
   but run all 5 at once with Promise.all)
3. Return: `ctx.send({ published, review, expired, rejected, banned })`

The 5 statuses and their filter clauses (copy from the existing `me()` switch-case):
- published: `{ user: userId, active: true, banned: false, rejected: false, remaining_days: { $gt: 0 } }`
- review: `{ user: userId, active: false, banned: false, rejected: false, remaining_days: { $gt: 0 }, $or: [{ is_paid: true, order: { $ne: null } }, { is_paid: false }] }`
- expired: `{ user: userId, active: false, banned: false, rejected: false, remaining_days: 0 }`
- rejected: `{ user: userId, rejected: true }`
- banned: `{ user: userId, banned: true }`

Implementation skeleton:
```typescript
async meCounts(ctx: any) {
  try {
    if (!ctx.state.user?.id) {
      return ctx.unauthorized("Debes estar autenticado para ver tus anuncios.");
    }
    const userId: number = ctx.state.user.id;

    const [published, review, expired, rejected, banned] = await Promise.all([
      strapi.entityService.count("api::ad.ad", { filters: { user: userId, active: true, banned: false, rejected: false, remaining_days: { $gt: 0 } } }),
      strapi.entityService.count("api::ad.ad", { filters: { user: userId, active: false, banned: false, rejected: false, remaining_days: { $gt: 0 }, $or: [{ is_paid: true, order: { $ne: null } }, { is_paid: false }] } }),
      strapi.entityService.count("api::ad.ad", { filters: { user: userId, active: false, banned: false, rejected: false, remaining_days: 0 } }),
      strapi.entityService.count("api::ad.ad", { filters: { user: userId, rejected: true } }),
      strapi.entityService.count("api::ad.ad", { filters: { user: userId, banned: true } }),
    ]);

    return ctx.send({ published, review, expired, rejected, banned });
  } catch (error) {
    return ctx.internalServerError("Internal server error");
  }
},
```

Do NOT modify any existing handlers. Only add the new route entry and the new method.
