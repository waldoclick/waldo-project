Related articles — same-category first, fallback to most recent:
```typescript
// 1. Load same-category articles (exclude current article):
await articlesStore.loadArticles(
  { categories: { slug: { $eq: article.categories[0]?.slug } }, slug: { $ne: article.slug } },
  { page: 1, pageSize: 6 },
  ['publishedAt:desc']
);
let related = articlesStore.articles;

// 2. If fewer than 3, top up with most recent (excluding current):
if (related.length < 3) {
  await articlesStore.loadArticles(
    { slug: { $ne: article.slug } },
    { page: 1, pageSize: 6 },
    ['publishedAt:desc']
  );
  // Merge: same-cat first, fill with most recent not already in list
  const ids = new Set(related.map(a => a.id));
  const fill = articlesStore.articles.filter(a => !ids.has(a.id));
  related = [...related, ...fill].slice(0, 6);
}
```
