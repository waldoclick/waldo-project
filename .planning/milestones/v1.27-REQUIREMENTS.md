# Requirements: Mostrar comprobante Webpay

## Milestone Objective
Display a Webpay (Transbank) digital receipt on /pagar/gracias immediately after a successful payment, with all mandatory fields visible, clear, and compliant with regulatory and user needs.

## Must-Have Fields (standard Webpay receipt in Chile)
- Transaction Amount (Monto)
- Authorization Code (Código de autorización)
- Payment Date/Time (Fecha y hora de pago)
- Payment Type (Tipo de pago)
- Last 4 Digits of Card (Últimos 4 dígitos)
- Order/Transaction Number (Número de orden/compra)
- Merchant Name or ID (Nombre/RUT/ID de comercio)
- Webpay/Transbank branding (logo present)

## Acceptance Criteria
- All above fields are always shown when payment is successful
- Data matches the record returned by Webpay/Transbank API (do not mask/change)
- Receipt is visible on desktop and mobile
- Labels are localized in Spanish (example: 'Monto', not 'Amount')
- If a required field is missing, display a warning or placeholder indicating data unavailable
- Receipt is only shown after a successful payment (never during, never if failed)

## Out of Scope
- Email receipts
- Support for non-Webpay gateways
- Printing/Download features
