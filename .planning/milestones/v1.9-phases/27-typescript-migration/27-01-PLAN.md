<objective>
Migrate all 18 remaining website pages from `<script setup>` to `<script setup lang="ts">`, and
eliminate all `any` type annotations in the 3 critical stores (`ad.store.ts`, `me.store.ts`,
`user.store.ts`) and 3 composables (`useAdAnalytics.ts`, `useAdPaymentSummary.ts`,
`usePackPaymentSummary.ts`).

Purpose: TS-01 ensures every page participates in TypeScript type checking. TS-02/TS-03 eliminate
silent runtime errors caused by untyped store state and composable functions. Together these are the
prerequisite for Phase 28 enabling `typeCheck: true` in the Nuxt build.

Output: 18 pages gain `lang="ts"`, 8 `any` usages removed from stores, 3 `as any` casts removed
from composables, `DataLayerEvent` interface defined inline in `useAdAnalytics.ts`.
</objective>

<task id="2" type="auto" name="Eliminate any in stores and composables (TS-02 + TS-03)">
Fix all `any` usages in the 3 target stores and 3 target composables.

### `apps/website/app/stores/ad.store.ts`
- `view_item_list: [] as any[]` → `[] as AnalyticsItem[]`
  (`AnalyticsItem` is already defined in the same file — no new import needed)
- `updateViewItemList(items: any[])` → `updateViewItemList(items: AnalyticsItem[])`

### `apps/website/app/stores/me.store.ts`
- Add import: `import type { User } from "@/types/user";`
- `const me = ref(null)` → `const me = ref<User | null>(null)`
- `saveUsername(data: any)` → `saveUsername(data: { username: string })`
  (the function body only uses `data.username` — the narrower type is exact)

### `apps/website/app/stores/user.store.ts`
- `loadUser(slug: any)` → `loadUser(slug: string)`
- `updateUserProfile(userId: string, userData: any)` → `updateUserProfile(userId: string, userData: Record<string, unknown>)`
  (`userData` is a partial user object that may also include `recaptchaToken` — `Record<string, unknown>` is the safe type)

### `apps/website/app/composables/useAdAnalytics.ts`
- Define a `DataLayerEvent` interface above the `pushEvent` function:
  ```ts
  interface DataLayerEvent {
    event: string;
    flow: string;
    ecommerce?: Record<string, unknown>;
  }
  ```
- Change `const eventData: any = {` → `const eventData: DataLayerEvent = {`

### `apps/website/app/composables/useAdPaymentSummary.ts`
- `const p = selectedPack.value as any;` → remove the cast entirely.
  `selectedPack.value` is already typed as `Pack | null`; the surrounding
  `if (selectedPack.value)` guard ensures it is non-null at that point.
  Replace references to `p.total_ads`/`p.price` with `selectedPack.value.total_ads`/
  `selectedPack.value.price` directly.

### `apps/website/app/composables/usePackPaymentSummary.ts`
- `const pack = selectedPack.value as any;` → remove the cast.
  Same pattern — replace `pack.price` references with `selectedPack.value.price`.

Done when: `grep -n ': any\|as any' apps/website/app/stores/ad.store.ts apps/website/app/stores/me.store.ts apps/website/app/stores/user.store.ts apps/website/app/composables/useAdAnalytics.ts apps/website/app/composables/useAdPaymentSummary.ts apps/website/app/composables/usePackPaymentSummary.ts` returns zero results.
</task>
