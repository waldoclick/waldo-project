<!-- Established Strapi SDK cast pattern (from prior phases) -->

Pattern for Strapi SDK response body (single item):
```typescript
const response = await strapi.findOne("resource");
const data = response as unknown as MyType;
```

Pattern for Strapi SDK filter params — SDK types do not accept arbitrary filter operators.
Use the double-cast pattern:
```typescript
const response = await strapi.find("resource", {
  filters: {
    field: { $eq: value },
  } as unknown as Record<string, unknown>,
  populate: "*",
});
```
