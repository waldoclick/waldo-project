<objective>
Add the two infrastructure prerequisites that Google Identity Services (GIS) One Tap requires: extend the website's Content Security Policy to allow GIS FedCM network calls, and add `GOOGLE_CLIENT_ID` to Strapi's environment.

Purpose: Without these changes, One Tap will fail silently — FedCM requests to `accounts.google.com/gsi/` are blocked by CSP before reaching the browser, and Phase 097's `OAuth2Client.verifyIdToken()` call has no client ID to verify against.

Output: Two targeted config edits — `nuxt.config.ts` (2 array entries added) and `apps/strapi/.env` (1 new line) + `.env.example` comment updated.
</objective>
