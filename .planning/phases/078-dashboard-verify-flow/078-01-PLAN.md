<objective>
Replace the `useStrapiAuth().login()` call in FormLogin.vue with a direct `useStrapiClient()` POST to `/api/auth/local`. When the backend returns `{ pendingToken, email }`, store the pendingToken in `useState('pendingToken')` (transient SSR-safe state) and navigate to `/auth/verify-code`.

Purpose: The Strapi backend (Phase 077) now returns `{ pendingToken, email }` instead of a JWT when `POST /api/auth/local` is called. Calling `useStrapiAuth().login()` which expects a JWT in the response will throw or fail silently. We must call the endpoint directly and handle the new response shape.

Output: Updated `FormLogin.vue` that hands off to the verify-code page instead of completing login directly.
</objective>

**Remove:**
- `const { login } = useStrapiAuth();` line
- The `await login({ ... })` call and all code that follows it inside the try block

**Add / replace the try block body with:**

```javascript
// Execute reCAPTCHA v3
const token = await $recaptcha.execute("submit");

// Call POST /api/auth/local directly — backend now returns { pendingToken, email }
// (useStrapiAuth().login() is NOT used because it expects a JWT, not a pendingToken)
const client = useStrapiClient();
const response = await client("/auth/local", {
  method: "POST",
  body: {
    identifier: values.email,
    password: values.password,
    recaptchaToken: token,
  },
});

// Store pendingToken in transient SSR-safe state, then navigate to verify page
const pendingToken = useState<string>("pendingToken", () => "");
pendingToken.value = response.pendingToken;
router.push("/auth/verify-code");
```

**Keep the catch block exactly as-is** — error messages for "Your account email is not confirmed" and "Invalid identifier or password" still apply since Strapi throws those before issuing pendingToken.

**Remove from imports:** `const { login } = useStrapiAuth()` — the `useStrapiAuth` import can remain if strapi is still used elsewhere, but since `strapi.find("users/me")` is no longer called here, also remove `const strapi = useStrapi();` if no longer needed.

- the `strapi.find("users/me", ...)` call
- the manager role check block
