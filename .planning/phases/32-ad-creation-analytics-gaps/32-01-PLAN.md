       flow entry.

    `adAnalytics.viewItemList(analyticsItems)` call inside `onMounted`:

    ```typescript
    // Fire step 1 once on initial mount (not on back-navigation — step persists in store)
    adAnalytics.stepView(1, "Payment Method");
    ```

    ```typescript

          3: "Personal Information",
          4: "Product Sheet",
          5: "Image Gallery",
        };
        adAnalytics.stepView(newStep, stepNames[newStep] ?? "");
      },
    );
    ```
    Note: No options object at all (remove the trailing `{ immediate: true }` entirely).
