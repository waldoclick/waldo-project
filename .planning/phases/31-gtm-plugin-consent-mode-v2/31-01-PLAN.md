<objective>
Fix the broken GTM plugin and implement Consent Mode v2 across two files in `apps/website`.

Purpose: GTM currently receives no valid dataLayer pushes because a local `gtag()` shim pushes JavaScript arrays instead of objects. GA4 page_view events are silently dropped on every SPA navigation. Additionally, there is no Consent Mode v2 default-denial push before GTM loads, which blocks GA4 data collection in EU/EEA by default under Google's Consent Mode requirements.

Output:
- `gtm.client.ts` — local shim removed; Consent Mode v2 default denial pushed before script injection; `router.afterEach` pushes a plain `page_view` object directly
- `LightboxCookies.vue` — `acceptCookies()` pushes the Consent Mode v2 update command instead of the custom `accept_cookies` event
</objective>
