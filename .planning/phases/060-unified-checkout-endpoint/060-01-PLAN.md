### `initiateCheckout(payload: CheckoutPayload, userId: string): Promise<InitiateResult>`

1. Validate that `pack` is not `"free"` and not `"paid"` — return `{ success: false, message: "Invalid pack" }` if so.
2. Look up the pack by name using `strapi.db.query("api::ad-pack.ad-pack").findOne({ where: { name: payload.pack } })` — return error if not found.
3. Validate that `price` is defined and not `"0"` — return `{ success: false, message: "Pack is not a paid pack" }` if zero price.
4. Build `amount = Number(packData.price)`.
5. If `payload.featured === true`, add `Number(process.env.AD_FEATURED_PRICE) || 10000` to amount.
6. Encode the buy_order: `const buyOrder = \`order-checkout-${userId}-${packData.id}-${payload.ad_id ?? 0}-${payload.featured ? 1 : 0}-${Date.now()}\``. SessionId: `const sessionId = \`session-checkout-${userId}-${Date.now()}\``.
7. Set `returnUrl = \`${process.env.APP_URL}/api/payments/webpay\``.
8. Call `getPaymentGateway().createTransaction(amount, buyOrder, sessionId, returnUrl)`.
9. If not successful, return `{ success: false, message: result.error ?? "Webpay transaction failed" }`.
10. Return `{ success: true, url: result.url, token: result.token }`.

### `processWebpayReturn(token: string): Promise<ProcessResult>`

Parse context from committed transaction buy_order. Follow the same pattern as `processPaidWebpay` in `ad.service.ts` but adapted to the new buy_order format.

1. Call `getPaymentGateway().commitTransaction(token)`.
2. If not successful or status !== "AUTHORIZED", return `{ success: false, message: "Payment not authorized" }`.
3. Parse `buyOrder = wepbayResponse.response.buy_order`. Format: `order-checkout-{userId}-{packId}-{adId}-{featured}-{timestamp}`.
   - Split on `-`, positions: [0]="order", [1]="checkout", [2]=userId, [3]=packId, [4]=adId, [5]=featured, [6]=timestamp
   - `const userId = parts[2]`
   - `const packId = Number(parts[3])`
   - `const adId = Number(parts[4])` (0 means no ad)
   - `const featured = parts[5] === "1"`
4. Look up pack: `PaymentUtils.adPack.getAdPack(packId)` — return error if not found.
5. Destructure `{ price, total_ads, total_days, total_features }` from `packData.data` — guard each with `??` defaults: `total_ads ?? 1`, `total_days ?? 30`, `total_features ?? 0`, `price ?? 0`.
6. Calculate `unitPrice = Number(price) / (total_ads ?? 1)`.
7. Create `total_ads` paid ad-reservations:
   ```typescript
   let firstAdReservationId: number | string | undefined;
   for (let i = 0; i < resolvedTotalAds; i++) {
     const res = await PaymentUtils.adReservation.createAdReservation(
       userId,
       unitPrice.toString(),
       resolvedTotalDays,
       "Reservation from unified checkout"
       // Do NOT pass adId here — link is made explicitly in step 7b below
     );
     if (i === 0 && res.success && res.adReservation) {
       firstAdReservationId = res.adReservation.id;
     }
   }
   ```
   **Step 7b** — explicitly link the first reservation to the ad (mirrors `ad.service.ts` lines 424-431):
   ```typescript
   if (firstAdReservationId !== undefined && adId > 0) {
     await PaymentUtils.ad.updateAdReservation(adId, firstAdReservationId);
   }
   ```
8. If `resolvedTotalFeatures > 0`, create `resolvedTotalFeatures` pack-included featured reservations. Link the **first one** to the ad when `adId > 0` and `featured` is false (i.e., the user is not buying a standalone paid featured — the pack's included slot covers it):
   ```typescript
   for (let i = 0; i < resolvedTotalFeatures; i++) {
     // Link first pack-included featured slot to the ad if present and not already covered by standalone featured
     const featuredAdId = (i === 0 && adId > 0 && !featured) ? adId : undefined;
     const featuredRes = await PaymentUtils.adFeaturedReservation.createAdFeaturedReservation(
       userId, "0", "Featured reservation from unified checkout", featuredAdId
     );
     // If linked, also call updateAdFeaturedReservation to make the association explicit on the ad record
     if (i === 0 && adId > 0 && !featured && featuredRes.success && featuredRes.adFeaturedReservation) {
       const reservationRecord = featuredRes.adFeaturedReservation as { id?: number };
       if (reservationRecord.id !== undefined) {
         await PaymentUtils.ad.updateAdFeaturedReservation(adId, reservationRecord.id);
       }
     }
   }
   ```
9. If `featured === true` (standalone paid featured), create ONE additional featured reservation and link it to the ad:
   ```typescript
   if (featured && adId > 0) {
     await PaymentUtils.adFeaturedReservation.createAdFeaturedReservation(
       userId,
       (Number(process.env.AD_FEATURED_PRICE) || 10000).toString(),
       "Paid featured from unified checkout",
       adId
     );
   }
   ```
10. If `adId > 0`:
    - Update ad dates: `await PaymentUtils.ad.updateAdDates(adId, resolvedTotalDays)`
    - Publish the ad: `await PaymentUtils.ad.publishAd(adId)`  ← **mandatory per v1.21 decision**
11. Zoho CRM sync (floating promise, non-blocking — same pattern as `processPaidWebpay` in `ad.service.ts`):
    ```typescript
    const _amount = wepbayResponse.response?.amount;
    Promise.resolve().then(async () => {
      if (!userId) return;
      // Need user email — load from strapi if adId is available
      // If adId > 0, get email from ad; otherwise skip (pack-only can't sync without user email)
      if (adId > 0) {
        const adData = await PaymentUtils.ad.getAdById(adId);
        const email = adData.ad?.user?.email;
        if (!email) return;
        const contact = await zohoService.findContact(email);
        if (!contact) return;
        const closingDate = new Date().toISOString().split("T")[0];
        await zohoService.createDeal({
          dealName: `Checkout Payment - ${adId}`,
          amount: _amount,
          contactId: contact.id,
          type: "Ad Payment",
          closingDate,
          leadSource: "Website",
        });
        await zohoService.updateContactStats(contact.id, { Total_Spent__c: _amount });
      }
    }).catch((err: { message?: string }) => {
      logger.error("Zoho sync failed for checkout — payment unaffected", { error: err.message });
    });
    ```
