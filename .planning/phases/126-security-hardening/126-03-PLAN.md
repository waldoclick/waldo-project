<objective>
Close the MEDIUM free-ad publish IDOR. Today `processFreeAd(adId, userId, pack)` loads the ad by id and then publishes it / links a reservation / updates its dates without verifying the ad belongs to `userId`. A user can pass another user's `adId` and publish or relink that ad. Add an ownership assertion (`ad.user.id === userId`, string/number-safe) immediately after `getAdById(adId)` and before any mutation (`updateAdReservation`, `updateAdDates`, `publishAd`); on mismatch, return a failure result equivalent to 403 (do not proceed).

Purpose: Ensure a user can only free-publish their own ad.
Output: The ownership check in free-ad.service.ts and a Jest regression test.
</objective>
