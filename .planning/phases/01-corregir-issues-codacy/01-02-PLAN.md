Scalar coercion (`Number()` / `String()`) is the closing fix.

Output: ad.ts saveDraft uses `Number(ad.ad_id)`; checkout.service uses `String(payload.pack)`; the Wave 0 ad_id and payload.pack injection tests go GREEN.
