    Replace with: `await apiClient(`/upload/files/${image.id}`, { method: "DELETE" })`.
    Note: Upload file deletion uses numeric `id`, not `documentId` — this is correct for the upload plugin.

    **Step 5 — Remove useStrapi if only used for mutations:**
    Check if `useStrapi()` is also used for reads (e.g., `strapi.findOne`, `strapi.find`). If yes, keep it. If only used for `strapi.update` and `strapi.delete`, remove it.
