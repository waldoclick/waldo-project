<objective>
Replace N-request drag-to-reorder with a single bulk POST /reorder endpoint for FAQs, policies, and terms. Also apply the dual-id (numeric id vs documentId) update fix from `faq.update()` to policy and term controllers so the existing PUT path works correctly too.

Purpose: Reordering a list of 20 items currently fires 20 PUT requests in parallel (network spam, race conditions, partial failures). One POST is atomic from the client's perspective and cleaner server-side. Policy/term custom update is required because the dashboard sends documentId strings while default core controllers expect numeric ids for direct db.query calls (though default core controllers handle documentId through the document service, our faq controller precedent documents the pattern and we need reorder either way).

Output: Three custom Strapi controllers + custom reorder routes + three dashboard components updated to call the new endpoint with a single request.
</objective>

```typescript
async update(ctx) {
  const { id } = ctx.params;
  const { data } = ctx.request.body;
  const numericId = Number(id);
  const isNumericId =
    Number.isInteger(numericId) &&
    numericId > 0 &&
    String(numericId) === String(id);
  const faq = isNumericId
    ? await strapi.db.query("api::faq.faq").update({ where: { id: numericId }, data })
    : await strapi.db.query("api::faq.faq").update({ where: { documentId: id }, data });
  return { data: faq };
}
```

Edit `apps/strapi/src/api/faq/controllers/faq.ts` — add a new `reorder` method alongside existing actions. Implementation:

```typescript
async reorder(ctx) {
  const { data } = ctx.request.body as {
    data?: Array<{ documentId?: string; order?: number }>;
  };
  if (!Array.isArray(data) || data.length === 0) {
    return ctx.badRequest("data must be a non-empty array");
  }
  // Validate every entry
  for (const entry of data) {
    if (!entry.documentId || typeof entry.documentId !== "string") {
      return ctx.badRequest("Every entry must have a string documentId");
    }
    if (typeof entry.order !== "number" || !Number.isFinite(entry.order)) {
      return ctx.badRequest("Every entry must have a numeric order");
    }
  }
  await Promise.all(
    data.map((entry) =>
      strapi.db.query("api::faq.faq").update({
        where: { documentId: entry.documentId },
        data: { order: entry.order },
      }),
    ),
  );
  return { data: { count: data.length } };
}
```
