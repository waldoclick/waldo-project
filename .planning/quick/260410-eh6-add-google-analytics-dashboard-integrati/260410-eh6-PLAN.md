  <action>
    1. In `google-analytics.types.ts`, add two new interfaces after existing ones:
       ```typescript
       export interface GA4SummaryMetric {
         current: number;
         previous: number;
         delta: number; // percentage change
       }
       export interface GA4Summary {
         sessions: GA4SummaryMetric;
         users: GA4SummaryMetric;
         bounceRate: GA4SummaryMetric;
         avgDuration: GA4SummaryMetric;
       }
       ```
       Update `IGoogleAnalyticsService` to add `getSummary(): Promise<GA4Summary>`.

    2. In `google-analytics.service.ts`, add `getSummary()` method to the class:
       - Import `GA4SummaryMetric` and `GA4Summary` from types.
       - Two parallel `runReport` calls via `Promise.all`:
         - Current period: last 28 days (`startDate` = 28 days ago, `endDate` = today).
         - Previous period: days -56 to -29 (`startDate` = 56 days ago, `endDate` = 29 days ago).
       - Both reports use NO dimensions (just totals), same 4 metrics: `sessions`, `totalUsers`, `bounceRate`, `averageSessionDuration`.
       - Extract single row from each response. Parse the 4 metric values.
       - For sessions and users: values are totals (sum). For bounceRate and avgDuration: values are already averages from GA4 when no dimensions are used.
       - Compute delta for each: `((current - previous) / previous) * 100`. Guard division by zero (if previous === 0, delta = current > 0 ? 100 : 0).
       - Return `GA4Summary` object with all 4 metrics.

    3. In `google-analytics.ts` routes, add a new route entry BEFORE existing routes:
       ```typescript
       {
         method: "GET",
         path: "/google-analytics/summary",
         handler: "google-analytics.getSummary",
         config: { policies: ["global::isManager"] },
       },
       ```

    4. In `google-analytics.ts` controller, add `getSummary` handler following the same pattern as `getStats`:
       ```typescript
       async getSummary(ctx: Context) {
         try {
           const service = new GoogleAnalyticsService();
           ctx.body = await service.getSummary();
         } catch (error: unknown) {
           const message = error instanceof Error ? error.message : String(error);
           return ctx.internalServerError(`Google Analytics error: ${message}`);
         }
       },
       ```

    No changes to `index.ts` needed — it uses `export *` so new types are automatically exported.
  </action>
