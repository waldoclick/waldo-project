**4. Agregar los handlers en el `<script setup>` (después de `handleBusinessTypeInput`, alrededor de línea 741, antes de `handlePhoneInput`):**

```js
const handleAddressNumberInput = (event) => {

  form.value.address_number = value;
  event.target.value = value;

const handleBusinessAddressNumberInput = (event) => {
  const value = String(event.target.value || "").slice(0, 5);
  form.value.business_address_number = value;
```

- El handler debe asignar `event.target.value` además del `form.value.*` para que el DOM refleje el valor truncado (sin esto, el usuario ve el 6º carácter momentáneamente).
- Seguir convenciones: English para código, Spanish para mensajes yup visibles al usuario.
