<objective>
Fix pagination not resetting to page 1 when search filter changes in OrdersDefault.vue.

Root cause: `useAsyncData` with a `queryKey` that omits `searchTerm`, combined with client-side `filteredOrders`. When `currentPage` is already 1, the query key does not change on new search → no re-fetch. Even when a re-fetch does fire, the server returns an unfiltered total, so `totalPages` is wrong and results bleed across pages.

Fix: Replace `useAsyncData` + `filteredOrders` with the `watch({ immediate: true }) + fetchOrders()` pattern used by `AdsTable.vue`. Server handles filtering via Strapi `$containsi`, pagination and totals come from the server response.
</objective>

**New fetchOrders function:**
```ts
const fetchOrders = async () => {
  try {
    loading.value = true;
    const section = settingsStore.orders;
    const [field, direction] = section.sortBy.split(":");

    const searchParams: Record<string, unknown> = {
      pagination: {
        page: section.currentPage,
        pageSize: section.pageSize,
      },
      sort: `${field}:${direction}`,
      populate: ["user", "ad"],
    };

    if (section.searchTerm) {
      searchParams.filters = {
        $or: [
          { "user.username": { $containsi: section.searchTerm } },
          { "ad.name": { $containsi: section.searchTerm } },
          { buy_order: { $containsi: section.searchTerm } },
        ],
      };
    }

    const response = (await apiClient("orders", {
      method: "GET",
      params: searchParams as unknown as Record<string, unknown>,
    })) as {
      data?: Order[];
      meta?: {
        pagination?: { page: number; pageCount?: number; total: number };
      };
    };

    allOrders.value = Array.isArray(response.data) ? response.data : [];
    const pagination = response.meta?.pagination;
    const total = pagination?.total ?? 0;
    const pageSize = section.pageSize;
    const pageCount =
      pagination?.pageCount ?? (Math.ceil(total / pageSize) || 1);
    paginationMeta.value = {
      page: pagination?.page ?? 1,
      pageSize,
      pageCount,
      total,
    };
  } catch (error) {
    console.error("Error fetching orders:", error);
    allOrders.value = [];
    paginationMeta.value = null;
  } finally {
    loading.value = false;
  }
};
```

**New computeds:**
